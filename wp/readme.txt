UnivaPayを使用して店舗でクレジットカード決済が可能です。

== Description ==

WooCommerceのUnivaPay支払いゲートウェイを使用して、ストアでVisa、MasterCard、American Express、Discover、JCB、Diners Clubを直接受け入れます。
